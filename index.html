Advanced Integrated Cyber-Physical Systems Lab

Aug 2018 - Jun. 2019

I joined Advanced Integrated Cyber-Physical Systems Lab's Autonomous Racecar project at UC Irvine as my senior design project this summer. As a student researcher under the project, I mainly used ROS as the interface to interact with the autonomous vehicle, such as launching and testing software on the NVIDIA Jetson TX2 module installed on the vehicle. Also, I implemented Google Cartographer using data collected from the IMU and LIDAR to map the vehicle's environment, and worked on pathfinding and obstacle avoidance algorithms using Python and C++.

Jul 2018 - May 2019

I interned at NetObjex, Inc. as an IoT engineer since this summer working on a product called "PiQube". I utilized Yocto Project to set up basic layers and recipes for NetObjex's embedded Linux distribution on a Raspberry Pi, and developed Python scripts to perform health check on the Raspberry Pi through the usage of classes, threads, and MQTT library. I designed and implemented the software architecture for "PiQube" for different communication protocols' client-server communication on IoT-enabled devices.

Microbiomechanics Laboratory

Autonomous Racecar

Autonomous Racecar is a project which aims to build an autonomous vehicle based on the F1tenth standards that can run by itself on any track, with integration of various software algorithms. The main interface/operating system used in the project is ROS or Robot Operating System, where all the software is tested and uploaded onto the autonomous vehicle. My main task is to implement Google Cartographer as our main Simultaneous Localization and Mapping (SLAM) algorithm and Adaptive Monte Carlo Localization (AMCL) to localize the vehicle, and assist my teammates in implementing pathfinding and obstacle avoidance algorithm.

Machine Learning for Disease Prediction

Machine Learning for Disease Prediction is an UROP-funded project that makes use of an interactice web application and different machine learning algorithms to display and predict patients' skin diseases based on the data collected about their symptoms/traits. 1kWh Challenge

1kWh Challenge is another project from CalPlug to educate and demonstrate to middle schoolers about the importance of saving energy and being aware about how much energy is consumed by electronics through the use of a web application. The demonstration of the project works like a game where the middle schoolers will either earn or deduce points based on how much energy they generated or consumed at each game station, and compare points with others at the end of the demonstration. My task was to assist in the completion of the web application using Javascript and the AngularJS framework, and to manage the workflow of the demonstrations.

Safe Passage Electronic Force Monitor

Safe Passage Electronic Force Monitor is a medical device project from CalPlug, an on-campus organization I have been part of since the winter of my freshmen year. The purpose of the project is to build a FDA-approved medical device prototype to be tested by the Department of Urology at UC Irvine that is capable of monitoring the amount of force being exerted on patients during urinary surgeries. I was initally an Android developer under the project who helped build the project's first native Android application using Java, and later became the project's team lead who both built the React Native version of the Android application and oversaw the logistics of the project until its completiton in the early Fall of 2018.
